# Images Folder

This folder contains static image assets for the application.

## Usage

Files in the `public` folder can be referenced in your code starting from the root `/`:

```jsx
<img src="/images/your-image.png" alt="Description" />
```

## Current Images

- `t-bank-logo.png` - T Bank logo used in the application header and sidebar

## Adding New Images

1. Place your image files in this folder
2. Reference them in your code using `/images/filename.ext`
3. Supported formats: PNG, JPG, JPEG, GIF, SVG, WebP

## Best Practices

- Use descriptive filenames (e.g., `company-logo.png` instead of `image1.png`)
- Optimize images before uploading to reduce file size
- Use appropriate formats:
  - PNG for logos and images with transparency
  - JPG for photographs
  - SVG for vector graphics and icons
